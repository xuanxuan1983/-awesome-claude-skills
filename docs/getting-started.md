# 开始使用 Awesome Claude Skills

## 什么是 Claude Skills？

Claude Skills 是可复用的功能模块，可以扩展 Claude Code CLI 的能力。

## 安装技能

```bash
# 克隆仓库
git clone https://github.com/你的用户名/awesome-claude-skills.git

# 进入技能目录
cd awesome-claude-skills/skills/某个技能

# 安装依赖
pip install -r requirements.txt
```

## 创建新技能

1. 复制模板
```bash
cp -r templates/basic skills/my-skill
```

2. 编辑 SKILL.md
3. 实现功能
4. 提交 PR
