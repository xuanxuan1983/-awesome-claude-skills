# 贡献指南

感谢你对 Awesome Claude Skills 的兴趣！

## 如何贡献

### 提交新技能

1. **Fork 本仓库**
2. **创建技能目录** `skills/your-skill-name/`
3. **必需文件**:
   - `SKILL.md` - 技能说明文档
   - `README.md` - 使用指南
   - 示例代码（如有）
4. **更新主 README** - 在相应分类下添加你的技能
5. **提交 PR**

### SKILL.md 格式

```markdown
# 技能名称

## 描述
简短描述这个技能做什么

## 安装
```bash
# 安装命令
```

## 使用
```bash
# 使用示例
```

## 配置
环境变量或配置文件说明

## 作者
- @你的GitHub用户名

## License
MIT
```

### 代码规范

- 使用清晰的命名
- 添加必要的注释
- 包含错误处理
- 提供使用示例

## 问题反馈

如有问题，请提交 [Issue](../../issues)。
